This Privacy Policy explains how Offline PDF Tools collects, uses, and protects user information when using the mobile application.

1. Information We Collect

The application may collect the following categories of information.

1.1 Information You Provide

The app allows users to open, edit, convert, or manage PDF documents stored on their device. These files are:

  • Selected by the user manually
  • Processed locally on the device
  • Not uploaded to any external server

Examples include: PDF documents, Images selected for PDF conversion, and Text entered for PDF generation.

The application does not store or transmit these files externally.

2. File Access and Storage

To provide its functionality, the app may request permission to access files on your device. These permissions allow the app to:

  • Open PDF files
  • Save processed files
  • Export converted documents
  • Share generated files

All file processing occurs locally on the device. No document content is transmitted to the developer or third-party servers.

3. Local Data Storage

The application may store limited data locally on the device using system storage such as SharedPreferences, App storage directories, and Cache storage.

This data may include: App settings, Tool preferences, Premium feature status, and Temporary processing files. This information remains only on the user's device.
