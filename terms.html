2. Application Purpose

elib PDF is an offline application designed to provide users with tools for managing and processing PDF documents.

The application includes features such as:

  • PDF merging with custom sequencing
  • PDF splitting (Range, Fixed, or Equal parts extraction)
  • PDF compression
  • Image to PDF conversion
  • PDF to Image extraction
  • PDF text extraction
  • PDF metadata editing (Title, Author, Subject, Keywords)
  • Watermarking (Text and Image)
  • Page numbering
  • Document organization (Rotate, Delete, Add pages)
  • Security features (Password protection and unlocking)

All document processing occurs locally on the user’s device.
