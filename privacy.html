1. Information We Collect

The application may collect the following categories of information.

1.1 Information You Provide

The app allows users to open, edit, convert, or manage PDF documents stored on their device.

These files are:

  • Selected by the user through the file picker interface
  • Processed locally on the device using embedded libraries (such as PDFBox)
  • Not uploaded to any external server

Examples include:

  • PDF documents for editing or processing
  • Images selected for PDF conversion
  • Text or images used for watermarking or metadata editing

The application does not store or transmit these files externally.

2. File Access and Storage

To provide its functionality, the app may request storage-related permissions depending on the Android version.

These permissions allow the app to:

  • Open and display PDF files
  • Save processed files (merged, compressed, or converted)
  • Access images for Image-to-PDF features
  • Share files using system sharing options

All file processing occurs locally on the device. No document content is transmitted to external servers.

3. Local Data Storage

The application may store limited data locally on the device using SharedPreferences and internal storage.

This data may include:

  • App settings and preferences
  • Premium feature status (Pro, Ads-free, or Batch access)
  • Temporary cache used during processing
  • Tool-related configurations (such as split ranges or watermark settings)

This information remains only on the user's device.

4. Advertising

The application uses Google AdMob to display advertisements, including rewarded ads.

AdMob may collect certain information automatically, such as:

  • Device information and Advertising ID (AD_ID)
  • Interaction with advertisements

For more information:

Google Privacy Policy: https://policies.google.com/privacy

Users can manage or reset their advertising preferences in device settings.

5. Analytics

The application uses Firebase Analytics to understand general usage patterns.

This may include:

  • Feature usage (e.g., most used tools)
  • Device type and operating system version
  • Session duration

This data is used only to improve app performance and user experience. No personally identifiable information is collected.

6. Crash Reporting

The application uses Firebase Crashlytics to detect and fix errors.

Crash reports may include:

  • Device information at the time of crash
  • App version details

No personal files, document content, or sensitive data are included in crash reports.

7. In-App Purchases and Rewards

The application offers premium features through Google Play Billing.

  • Payments are securely handled by Google Play
  • The app does not store payment or card information
  • Rewarded ads may provide temporary access to premium features

8. Third-Party Services

The application integrates the following services:

  • Google AdMob (Advertising)
  • Firebase Analytics (Usage insights)
  • Firebase Crashlytics (Crash reporting)
  • Google Play Billing (Purchases)

9. Data Security

The application is designed with an offline-first approach. All document processing occurs locally on your device.

Your files are not uploaded, stored, or shared with external servers by the application.

10. Children's Privacy

This application is not intended for children under the age of 13. We do not knowingly collect personal information from children.
